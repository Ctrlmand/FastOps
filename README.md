# FASTOPS #001.5

A simple addon for Blender.

## Features
### View 3D > N > FASTOPS
* Rename Utility
* Batch Add & Clear Split Normal
* Batch Clear Sharp
* Select Objects By Name
* Quick Add Modifiers (F)
### Shader Editor > N > FASTOPS
* Merge RGB And A Texture Node In One Node
* Set Color Space In Common Use (F)
